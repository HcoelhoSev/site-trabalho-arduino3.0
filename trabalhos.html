Arduino Uno - Práticas

Semáforo Funcional

Foi desenvolvido um projeto no qual foi utilizado o arduino para criarmos um procedimento eletrônico, interativo e independente como também utilização de outros componentes para realizar a interação do circuito, sendo eles; LEDS, Protoboard e cabos. O arduíno é uma plataforma de desenvolvimento de projetos eletrônicos e tecnológicos, utilizando hardware e software, nele contém o microcontrolador que seria o ´´cerébro`` do arduíno. Já o LED é um dispositivo semicondutor que converte energia elétrica em luz visível. A Protoboard seria uma placa que serve como um protótipo de um aparelho eletrônico, que possibilita construir circuitos de teste com segurança e agilidade.

A data do desenvolvimento se deu início no dia vinte de Agosto de 2024. A princípio, foi elaborado um pequeno circuito utilizando LEDS semelhante a um semáforo, que contiveram as cores vermelho, amarelo e verde, nesse circuito foi criado um código para que as luzes ligassem de modo sequencial, cada cor ficaria ligada 15 segundos na ordem vermelho, amarelo, verde, de forma simultânea. Para a execução desse circuito foi utilizado o software Arduino IDE, para a aplicação do código, após finalização do mesmo, ele foi carregado para o arduíno, desse modo, realizando o funcionamento dos LEDS.

Led Controlador por Potênciometro

Foi desenvolvido um projeto que utilizou o Arduino para criar um procedimento eletrônico interativo e autônomo, além de empregar outros componentes para a interação do circuito, como LEDs, potenciômetro, Protoboard e cabos. O Arduino é uma plataforma destinada ao desenvolvimento de projetos eletrônicos e tecnológicos, integrando hardware e software, sendo o microcontrolador o "cérebro" do dispositivo. O LED, por sua vez, é um semicondutor que transforma energia elétrica em luz visível. A Protoboard funciona como uma placa protótipo para dispositivos eletrônicos, permitindo a construção de circuitos de teste de maneira segura e eficiente. Já o potenciômetro é um componente eletrônico com resistência ajustável.

O projeto 2 se deu início no dia vinte e sete de Agosto de 2024. O projeto consiste em potencializar a intensidade da luz de um único led. Para que isso fosse possível, desenvolvemos um código e criamos o circuito na protoboard, utilzando tanto o potenciômetro quanto o led para a interação desse processo, feito o código, realizamos o upload para o arduíno fazer a execução desse procedimento.
